# BridgeNode 项目主计划

## 项目概览

BridgeNode 是一个 SSH 隧道 Web 交互中间件，提供本地-集群数据交换、实时监控、文件传输等功能。

### 核心功能
- **LaTeX 编辑器** - 实时渲染 LaTeX 公式
- **文件传输** - 拖拽上传/下载，支持大文件分片
- **命令控制台** - 发送控制命令到远程服务器
- **日志追踪** - 实时查看远程日志文件
- **WebSocket 监控** - 实时隧道状态监控

### 技术栈
- **后端**: FastAPI + Uvicorn
- **前端**: 原生 HTML/CSS/JavaScript
- **WebSocket**: 原生 WebSocket
- **认证**: Token-based
